The following is a set of guidelines for contributing to Armada Den. These are mostly guidelines, not rules. Use your best judgment, and feel free to propose changes to this document in a pull request.

## How to Contribute

### Reporting Bugs

This section guides you through submitting a bug report.
- **Use a clear and descriptive title** for the issue to identify the problem.
- **Describe the exact steps which reproduce the problem** in as much detail as possible.
- **Provide specific examples** to demonstrate the steps.

### Pull Requests

1. **Fork the repo** and create your branch from `main`.
2. **Clone the project** to your own machine.
3. **Commit changes** to your own branch.
4. **Push your work** back up to your fork.
5. **Submit a Pull request** so that we can review your changes.

### Code Style

- We use **Python 3.12+**.
- Please ensure your code follows standard Python conventions (PEP 8).
- Run explicit tests for your changes where possible.
